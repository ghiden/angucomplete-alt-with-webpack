# Webpack and Angucomplete-alt

This is a very minimum set up for angucomplete-alt with webpack.

```bash
.
├── README.md
├── app
│   ├── bundle.js
│   ├── core
│   │   ├── bootstrap.js
│   │   └── vendor.js
│   ├── index.html
│   └── index.js
├── package.json
└── webpack.config.js
```

```bash
npm install
npm start
```
